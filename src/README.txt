Posix Lab

Utiliza fork para executar os testes de maniura isolada, evitando interferências de um teste no outro.
Para executar todos os testes $ ./exa.
Para executar um unico teste basta escrever $ ./exa testx, onde o x deve ser mudado para o número do teste que deseja executar.

O relatório de teste imprime os resultados seguindo um código de cores:
 - [PASS]: Verde;
 - [FAIL]: Vermelho;
 - [ERRO]: Magenta;
